# Modern Calculator

A sleek and modern calculator application with a glassmorphism design built with HTML, CSS, and JavaScript.

## Features

- **Basic Operations**: Addition, subtraction, multiplication, and ision
- **Modern UI**: Dark theme with glassmorphism effect and smooth animations
- **Responsive Design**: Centered layout with smooth button interactions
- **Keyboard Support**: Use your keyboard for quick calculations
- **Error Handling**: Displays error message for invalid expressions
- **Delete & Clear**: Remove the last character or clear the entire display

## Keyboard Shortcuts

| Key | Action |
| --- | --- |
| Numbers (0-9) | Append number to display |
| `+`, `-`, `*`, `/` | Append operator |
| `.` | Append decimal point |
| `Enter` | Calculate result |
| `Backspace` | Delete last character |
| `Escape` | Clear display |

## How to Use

1. Open `index.html` in your web browser
2. Click buttons to enter numbers and operators, or use your keyboard
3. Press `=` or `Enter` to calculate the result
4. Use `C` to clear the display or `⌫` to delete the last digit

## File Structure

- `index.html` - Main HTML file
- `style.css` - Styling
- `script.js` - Calculator logic

## Technologies Used

- **HTML5** - Structure
- **CSS3** - Styling with gradient backgrounds and backdrop filters
- **JavaScript** - Calculator logic and interactivity

## Design Features

- Glassmorphism effect with semi-transparent backdrop blur
- Gradient background with dark blue theme
- Smooth button hover and active states
- Color-coded buttons (operators in blue, equals in green, clear in purple)

## Browser Compatibility

Works in all modern browsers that support:

- CSS Grid
- Backdrop Filter
- CSS Gradients
- ES6 JavaScript
